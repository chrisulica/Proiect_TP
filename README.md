SIMULATOR DE SAH

Un simulator de sah, scirs in limbajul C, cu interfata grafica realizata cu raylib. 
Jocul suporta atat modul singeplayer (vs Ai) cat si modul pentru doi jucatori local.

Pentru a rula, utilizatorul trebuie sa aiba instalat: GCC, Make, raylib.

INSTALARE SI COMPILARE:

1. CLONARE REPOSITORY

git clone https://github.com/chrisulica/Proiect_TP/git

cd Proiect_TP/Proiect

3. COMPILARE

make

4. RULARE:

./chess
